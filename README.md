# Интеллектуальный Нормоконтролёр ВКР ИГУ

Система автоматической проверки выпускных квалификационных работ на соответствие требованиям ИГУ.

## Архитектура

Система состоит из двух контуров:

### Контур А: Детерминированный валидатор форматирования
- Проверка структуры документа
- Валидация форматирования (шрифты, поля, отступы)
- Проверка оформления таблиц, рисунков, формул
- Валидация библиографических ссылок

### Контур Б: Семантический анализ (LLM/RAG)
- Анализ содержания работы
- Проверка логической связности
- Выявление плагиата
- Оценка соответствия теме

## Установка

```bash
# Установка основных зависимостей
pip install -e .

# Установка дополнительных модулей
pip install -e ".[llm]"    # Для семантического анализа
pip install -e ".[api]"    # Для API сервера
pip install -e ".[dev]"    # Для разработки
```

## Быстрый старт

```bash
# Запуск проверки документа
python -m src.cli check document.docx

# Запуск тестов
pytest

# Запуск тестов с покрытием
pytest --cov=src
```

## Разработка (TDD)

Проект разрабатывается по методологии TDD:

1. **Красный этап**: Написание теста
2. **Зелёный этап**: Минимальная реализация для прохождения теста
3. **Рефакторинг**: Улучшение кода при сохранении проходящих тестов

```bash
# Запустить конкретный тест
pytest tests/test_rules.py -v

# Запустить все тесты
pytest

# Запустить с отчётностью о покрытии
pytest --cov=src --cov-report=html
```
